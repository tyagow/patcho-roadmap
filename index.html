Pra Jaluza — como continuar

A regra do projeto

O conteúdo de verdade está nos arquivos .md (texto). O HTML é só a apresentação bonita, gerada a partir do texto. Pra mudar a marca ou o plano, edite o .md — depois é só regenerar o HTML.

Dentro do kit (PATCHO-Kit.zip):

  • brand/ — a marca canônica: história, identidade, voz. Edite aqui.
  • roadmap/roadmap.md — o plano de lançamento, em texto editável.
  • research/ — a pesquisa que sustenta cada decisão (1 arquivo por tema).
  • brand-kit/ — os guias visuais e os editores de logo/selo/lip.
  • export/ — os logos prontos em SVG / PDF / PNG (8 versões) pro pessoal de design.

Qualquer editor de texto abre os .md. Os .html abrem em qualquer navegador (só precisam de internet pras fontes). Born in the yard. Built on dirt. 🤎
